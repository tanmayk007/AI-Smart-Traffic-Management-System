# 🚦 AI-Based Smart Traffic Management System

A mini-project developed in C language for smart traffic management and congestion analysis. The system uses traffic density and emergency conditions to optimize signal timings and improve traffic flow in smart-city environments.

---

## ✨ Features

- Adaptive traffic signal timing
- Emergency vehicle priority system
- Real-time congestion detection
- Add, update, search, and delete junction records
- Sort junctions by traffic density
- Daily traffic report generation
- Persistent file storage using binary files
- Login authentication system
- Traffic statistics dashboard
- Menu-driven console interface
